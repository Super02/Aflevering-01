2) Pde: Hvad er svaret på følgende kommando:
		println( 3.3 == 1.1 + 1.1 + 1.1 );
	- Kan du forklare resultatet ?
Svaret er falsk. Grunden til at computeren tror at 1.1 + 1.1 + 1.1 ikke er lig med 3.3 er på grund af noget som hedder floating point error og er en fejl som opstår når computeren arbejder med decimaler.
Processing mener at svaret er: 3.3000002

3) Java: Upload dit 'Tjek' program til dit GitHub repository.
	- Hvad mener du er årsagen til at 
		println( 0.3 == 0.1 + 0.1 + 0.1 );
          giver svaret false ?
Svaret er lidt det samme som med processing. Der sker igen en floating point error java mener dog at svaret er 0.30000000000000004. Og det er på grund af at Processing bruger 32 bit og java bruger 64 bit

Dine svar til følgende opgaver skal indsættes i en fil: Svar-01.txt
og denne fil skal uploades til GitHub repository.

4) Angiv seks 'CMD prompt' kommandoer du synes er vigtige,
   med en beskrivelse af hvad de gør.
   
   CD - Change directory bruges til at ændre hvilken mappe som du er i og kan udføre kommandoer ud fra
   MKDIR - Make directory bruges til at lave en ny mappe i den mappe som du er i
   DIR - Bruges til at vise de mapper som er i den mappe du er i
   COPY - Bruges til at kopiere filer
   KILL - Bruges til at slukke for processer som windows kører
   MV - Bruges til at flytte filer rundt
